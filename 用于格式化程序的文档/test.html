key 类型 描述
count int 单页条数
start int 数据的开始项
total int 数据总条数
subjects json数组 电影列表
— id string 电影id
— title string 电影名中文名
— images json对象 存放各种大小的电影图
—— small／large／medium string 电影图url
— genres json数组 电影类型
— rating json对象 评分信息
—— average float 电影评分
— directors json数组 导演列表
—— name string 导演名
— casts json数组 主演列表
—— name string 主演名
— year int 年份
